# Super Mr. Bennet
A 2D platformer game based on the events of Jane Austen's *Pride and Prejudice* and built in my Phoenix engine.

The game can be found [here](https://openchanter.github.io/super-bennet)!

The game features Mr. Bennet on his journey to rescue his daughter Lydia from the clutches of the evil Mr. Wickham!
There are a few levels as Mr. Bennet goes to different places around England, questioning many different people on the whereabouts of Wickham and his daughter, each with their own challenges to beat and their own segment of the story!

The art for the project is all made in Aseprite, and the game is written in Typescript using a custom game [engine](https://github.com/OpEnchanter/phoenix) and runs in any modern browser.
